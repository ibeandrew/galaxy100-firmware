# Flashing the Galaxy 100

1. Put the keyboard in DFU mode (unplug, then hold the PCB reset
   button OR hold Esc while plugging in).
2. From this folder run:

       .\wb32-dfu-updater_cli\wb32-dfu-updater_cli.exe -D epomaker_galaxy100_default.bin

3. Wait for "Done!". Unplug and replug the keyboard (out of DFU mode).
